# Titanic Survival Prediction – Data Science Mini Project

## Project Overview

This project analyzes the Titanic dataset and builds machine learning models to predict whether a passenger survived or not. The analysis includes data cleaning, exploratory data analysis (EDA), model training, evaluation, and comparison of different machine learning models.

---

## Objective

The objective of this project is to apply data science techniques to:

* Understand the Titanic dataset
* Perform data preprocessing and cleaning
* Explore patterns using visualizations
* Train machine learning models
* Evaluate and compare model performance

---

## Dataset Information

The dataset contains **891 passenger records** with information such as:

* PassengerId
* Pclass (Passenger Class)
* Sex
* Age
* SibSp (Siblings/Spouses aboard)
* Parch (Parents/Children aboard)
* Fare
* Embarked
* Survived (Target Variable)

The target variable **Survived** indicates whether a passenger survived the Titanic disaster.

## Machine Learning Models Used

### Logistic Regression

A classification algorithm used to predict passenger survival.

### Random Forest Classifier

An ensemble learning algorithm used to improve prediction accuracy and analyze feature importance.

---

## Model Evaluation

The models were evaluated using:

* Accuracy Score
* Confusion Matrix
* Visualization graphs

Logistic Regression achieved approximately **80% accuracy** on the test dataset.

---

## Feature Importance

Random Forest was used to determine which features influence survival prediction the most.
Important features include:

* Passenger Class (Pclass)
* Sex
* Fare
* Age
